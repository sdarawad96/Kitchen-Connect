# Kitchen Connect

Kitchen Connect is a recipe-sharing website where users can explore and share recipes from different cultures, diets, and regions.

## Features
- Browse recipes by category
- View featured and popular recipes
- Submit your own recipes
- Simple and user-friendly design

## Technologies Used
- HTML
- CSS

## How to Use
- Open the website
- Navigate using the menu (Home, Browse, Submit, About)
- Explore recipes or submit your own
